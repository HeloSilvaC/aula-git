git init -> cria uma pastinha .git

code . -> abre o vs

git add . -> adiciona todos os arquivos na pasta git

git commit -m "criando teste.txt" -> descrever bem detalhado

cls -> limpa

git checkout 1754b987104d069a0c508bcc256e8bf8eba3b57f -> volta para o commit anterior

git checkout -> selecionar a branch

git remote add origin git@github.com:HeloSilvaC/aula-git.git -> adicionar o repositório remoto

git push origin master -> subir no repositório local no origin que foi criado

master -> branche principal

branchs -> ramos/ramificações

git branch -> mostra as branches

git branch teste1 -> cria a branch

git merge teste1 -> pega o que fiz no teste um e manda para o master pq já fui para ele através do git checkout master
